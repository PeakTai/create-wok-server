# wok-ui 默认模板

这是一个基于 wok-ui 的默认模板，使用 vite 构建，用于快速启动和开发 wok-ui 应用的原型。

## 目录结构

```
├── public -----------------   静态资源目录，包含了应用的静态资源文件。
├── src --------------------   源代码目录，包含了应用的源代码文件。
│   ├── api ----------------   后端 API 接口对接
│   ├── i18n ---------------   国际化配置文件
│   ├── modules ------------   项目扩展的通用模块
│   ├── pages --------------   页面模块文件，定义了应用的页面模块。
│   ├── utils --------------   工具函数文件，定义了应用的工具函数。
│   ├── main.ts ------------   应用入口文件，负责初始化路由和挂载应用到 DOM 中。
│   └── router.ts ----------   路由规则文件，定义了应用的路由规则。
├── index.html -------------   vite 应用入口 HTML
├── package.json -----------   项目依赖配置文件，定义了项目的依赖关系和构建配置。
├── README.md --------------   项目说明文档，包含了项目的介绍、目录结构和使用说明。
├── tsconfig.json ----------   TypeScript 配置文件，定义了项目的 TypeScript 编译选项。
└── vite.config.ts ---------   vite 配置文件
```
